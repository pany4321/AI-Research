LLaMA-Factory 内部分析

LlamaBoard Web UI 三层架构

Engine → Manager → Runner

E Engine 顶层控制器，管理全局状态
M Manager 组件注册表 ID ↔ Gradio
R Runner 训练子进程生命周期管理

Runner 训练流程

验证参数 → 解析参数 → 保存配置 → 设置环境变量 → Popen 启动

数据处理系统

DataLoader：dataset_info.json → datasets.load_dataset() → 格式对齐 → 对话模板

对话模板：llama2、llama3、qwen、chatml、mistral 等

Data Collator：Padding · Label Masking · 序列打包

模型系统 & 配置

模型加载器 ModelLoader

AutoModel.from_pretrained() → 量化 → device_map → torch_dtype

适配器注入与补丁

AdapterInjector：LoraConfig → get_peft_model()

Patcher：FlashAttention-2 · S² Attention · Liger Kernel

模型注册表

管理 100+ 模型的支持信息（架构、模板、LoRA 目标模块）
