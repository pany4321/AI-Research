设计理念

「配置即代码」

将复杂的 Python 训练脚本转化为声明式 YAML 配置，无需编写训练脚本即可启动微调。

GitHub 11.7k+ Stars · 企业级分布式训练支持

YAML 配置示例

base_model: Qwen/Qwen2.5-7B
adapter: qlora
lora_r: 32
datasets: [{path: train.jsonl, type: alpaca}]
micro_batch_size: 2
gradient_accumulation_steps: 4

六阶段训练流水线

1配置加载 → YAML 解析 + Pydantic 验证
2环境设置 → FSDP / DeepSpeed + 混合精度
3数据集加载 → 分词/过滤/打包/缓存
4模型加载 → 量化 + FlashAttn + 适配器
5训练器构建 → 参数 + 优化器 + 回调
6训练与保存 → 检查点 + 适配器导出

企业特性

训练器 & 回调系统

后训练器回调：BenchEval / EarlyStopping / ReLoRA / LISA
